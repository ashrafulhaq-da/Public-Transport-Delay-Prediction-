 
🚍 Public Transport Delay Prediction

📌 Overview

This project aims to predict whether a public transport vehicle (bus/train) will be delayed based on real-world factors such as weather conditions, traffic congestion, and time-related features.

The goal is to build a machine learning model that can help improve transportation planning and decision-making.



🎯 Objectives

* Predict transport delays using machine learning
* Analyze key factors affecting delays
* Perform data cleaning and preprocessing
* Build and evaluate classification models


 📊 Dataset

The dataset includes multiple features such as:

* Weather conditions
* Traffic levels
* Time of day
* City events

 Dataset link:https://www.kaggle.com/datasets/khushikyad001/public-transport-delays-with-weather-and-events



🛠️ Technologies Used

* Programming: Python
* Libraries: Pandas, NumPy, Scikit-learn
* Visualization: Matplotlib, Seaborn
* Environment:Jupyter Notebook



 🔍 Workflow

1. Data Collection & Understanding**

   * Loaded dataset and explored structure

2. Data Cleaning & Preprocessing**

   * Handled missing values
   * Encoded categorical variables
   * Normalized/processed features

3. Exploratory Data Analysis (EDA)**

   * Visualized trends and relationships
   * Identified patterns in delays

4. Model Building**

   * Trained classification models (e.g., Logistic Regression, Decision Tree)

5. Model Evaluation**

   * Evaluated performance using accuracy and other metrics
   * Achieved approximately **80% accuracy**

6. Feature Importance Analysis**

   * Identified key factors influencing delays



📈 Results

* Built a predictive model with ~80% accuracy
* Found that weather and traffic are major contributors to delays
* Generated insights that can support better transport planning
